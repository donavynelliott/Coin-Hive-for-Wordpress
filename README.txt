This plugin adds a couple options for monetizing your website via your website guests CPU power.

== Description ==

Monetize Your Business With Your Users' CPU Power

This plugin introduces methods for integrating Coin-Hive.com's proprietary javascript mining software into your current website.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `coin-hive.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
