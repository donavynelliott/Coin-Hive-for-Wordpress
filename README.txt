This plugin adds a new option for monetizing your website via your website guests CPU power.

== Description ==

Monetize Your Business With Your Users' CPU Power

This plugin introduces methods for integrating Coin-Hive.com's proprietary javascript mining software into your current website. It currently allows you to warn users, show them stats, and auto start the mining software.

== Installation ==

1. Upload `coin-hive.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Insert your public key through the 'Coin Hive' menu in WordPress
4. Choose your options through 'Coin Hive' -> 'Background Mining' menu
5. Insert the shortcode [coin_hive_button text="open stats"] or through html
    <a href="#"><span id="coin-hive-show" onclick="openHiveUI()">Open Stats</span></a>

== Frequently Asked Questions ==

Q. Will visitors know what is happening?
A. There are options in the admin menu to warn them.

Q. Will this work with my theme?
A. The css is explicit as to prevent conflicts between other themes. The button to open the stats UI is through a shortcode.
